# Phoenix-Project
Mass repository for other Linux and BSD Distrubutions to integrate with BlackArch
Know that some of these Distrubutions are outdated and have to be updated.
Some of the Distros included are: Live Hacking OS, Anonymous OS, Bugtraq-II Black Widow, ArchStrike, Kali Linux, Kali Linux Purple, Pentoo etc
For BlackArch documentation, refer to https://blackarch.org/docs/html
For official BlackArch source code, refer to https://github.com/BlackArch
